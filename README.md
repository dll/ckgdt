# CKGDT 课程知识图谱与数字孪生平台

CKGDT（Course Knowledge Graph and Digital Twin Platform）是一个基于 Flutter 的多课程教学平台，面向课程知识图谱、学习过程数字孪生、实验/考核/作品评价、OBE 达成度分析和 AI 多智能体协作场景。

## 核心能力

- 多课程知识图谱：支持课程目标、知识点、资源、测验和学习路径关联。
- 教学数字孪生：聚合学生学习、实验、考核、作品、平时成绩和达成度数据。
- 全流程评价：覆盖课堂、学习、实验、考核、作品、达成、归档等教学环节。
- AI 协作：18 个专业智能体支持问答、导航、批阅辅助、归档、课程生成和质量审查。
- 多端运行：支持 Windows、Android、Web、HarmonyOS 等 Flutter 目标平台。

## 技术栈

- Flutter / Dart
- sqflite / sqlite3
- Material Design 3
- Gitee/GitHub 同步与发布
- LLM API、RAG 和多智能体编排
