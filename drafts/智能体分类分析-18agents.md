# 18 个智能体分类与功能全景

## 重构回顾：25 → 18

| 合并组 | 旧 Agent | 新 Agent | 合并方式 |
|--------|---------|----------|---------|
| 学习导师 | learning + path + tutor | **tutor** | 3 合 1（辅导/笔记/路径三模式） |
| 批阅官 | lab_grading + assessment_grading + works_grading | **grading** | 3 合 1（12 DAO 工具统一批阅） |
| 课程管家 | courseware + course_gen | **courseware** | 2 合 1（课件+一键生课） |
| 数字孪生 | virtual_student + virtual_teacher | **digital_twin** | 2 合 1（学生/教师双模式） |
| 通用助手 | assistant + madkg | **assistant** | 2 合 1（通用问答+系统指引） |

---

## 一、教学辅导类（4 个）

面向学生的直接学习支持——答疑、讲解、练习、思政融入。

### 1. 🎓 学习导师（tutor）
| 维度 | 说明 |
|------|------|
| **来源** | learning + path + tutor 三合一 |
| **三模式** | 📖 辅导（费曼+苏格拉底）、📝 笔记（5 种笔记类型）、🗺️ 路径（ADDIE+最近发展区） |
| **能力增强** | `useRag: true` — 基于课程知识库 RAG 检索增强 |
| **角色** | 不限 |

### 2. 📱 移动专家（mobile_expert）
| 维度 | 说明 |
|------|------|
| **专注领域** | 6 大技术栈（Android/iOS/Flutter/React Native/小程序/HarmonyOS） |
| **能力增强** | `useRag: true` — 6 大技术栈课程知识库 RAG |
| **特色** | 多维技术对比、≤20 行代码示例、从零搭项目实战指导 |
| **角色** | 不限 |

### 3. 🏛️ 思政伦理（ethics）
| 维度 | 说明 |
|------|------|
| **五大领域** | 家国情怀、职业伦理、法律法规、安全开发、AI 伦理 |
| **特色** | 结合移动开发实际案例，不空洞说教 |
| **角色** | 不限 |

### 4. 📝 测验教练（quiz）
| 维度 | 说明 |
|------|------|
| **核心能力** | 出题（三级难度⭐~⭐⭐⭐）+ 智能批改 + 错题分析（按知识点聚类） |
| **工具** | `get_my_wrong_answers`、`get_my_quiz_summary`、`get_chapter_question_stats` |
| **能力增强** | `useRag: true` |
| **角色** | 不限 |

---

## 二、内容生成类（3 个）

自动化产出教学文档、课件、归档材料。

### 5. 📚 课程管家（courseware）
| 维度 | 说明 |
|------|------|
| **来源** | courseware + course_gen 二合一 |
| **双重能力** | 课件生成（教案/PPT/UML/视频脚本）+ 一键生课（新课程 8 要素全链路） |
| **方法论** | ADDIE 模型 + 布鲁姆分类 |
| **角色** | 不限 |

### 6. 📄 文档转换（doc_converter）
| 维度 | 说明 |
|------|------|
| **格式矩阵** | 自然语言 → Markdown → PPT/PDF/报告模板/项目文档/会议纪要 |
| **模板库** | 实验报告五段式、项目文档、技术方案 |
| **角色** | 不限 |

### 7. 📦 归档助手（archive）
| 维度 | 说明 |
|------|------|
| **覆盖阶段** | 期初/期中/期末 全教学阶段 |
| **文档类型** | 18+：教学任务书/大纲/进度表/教案/课件/点名册/试卷/成绩分析/课程总结/审核表… |
| **数据来源** | 直接查询 DB（syllabus_items/lesson_plans/achievement_scores 等） |
| **限制** | 仅教师/管理员 |

---

## 三、评价批阅类（5 个）

自动化评分、成绩分析、达成度追踪。

### 8. 📝 批阅官（grading）
| 维度 | 说明 |
|------|------|
| **来源** | lab_grading + assessment_grading + works_grading 三合一 |
| **工具数** | **12 个** — 项目中最多的 DAO 工具 |
| **三种批阅** | 实验报告（5 维度）、考核报告（5 维度）、学生作品（两步评分） |
| **评分流程** | 真实性校验 → 五维度评分 → 标准化 JSON 输出 |
| **限制** | 仅教师/管理员 |

### 9. 📊 考核助理（assessment）
| 维度 | 说明 |
|------|------|
| **核心信息** | 分组/答辩/成绩查询 |
| **工具** | `list_assessment_groups`、`get_group_stats`、`get_score_overview` |
| **成绩构成** | 平时 15% + 实验 30% + 项目 35% + 测验 20% |
| **角色** | 不限 |

### 10. 🏆 达成分析师（achievement）
| 维度 | 说明 |
|------|------|
| **方法论** | OBE 达成度评价体系 |
| **4 个课程目标** | CO1~CO4，达成度 = Σ(环节成绩 × 环节权重) / 满分 |
| **工具** | `list_achievement_batches`、`get_class_achievement`、`get_improvement_suggestions` |
| **等级** | 优秀 ≥0.85 / 良好 ≥0.70 / 中等 ≥0.60 / 未达成 <0.60 |
| **角色** | 不限 |

### 11. 🎨 作品展评官（works）
| 维度 | 说明 |
|------|------|
| **评分维度** | 功能 25 / 技术 20 / 跨框架 25 / 性能 15 / 文档 15 |
| **等级标准** | S/A/B/C/D |
| **特色** | 先肯定再提改进，具体可操作 |
| **角色** | 不限 |

### 12. 🔬 实验助手（lab）
| 维度 | 说明 |
|------|------|
| **覆盖** | 6 次实验对应 6 章 |
| **工具** | `list_lab_tasks`、`get_my_lab_submissions` |
| **评分标准** | 功能 40% + 代码质量 30% + 文档 20% + 创新 10% |
| **迟交政策** | 每天扣 10%，超 5 天不接受 |
| **角色** | 不限 |

---

## 四、交互入口与辅助支撑（6 个）

系统级入口、导航、工具与后台监控。

### 13. 🎙️ 语音助手（voice）
| 维度 | 说明 |
|------|------|
| **定位** | 全局语音/自然语言导航入口 |
| **4 层路由** | 快速路径（返回/退出）→ Tab 映射 → 子页面匹配 → AI 意图兜底 |
| **输出** | 严格 JSON 格式导航指令 |
| **角色** | 不限 |

### 14. 🤖 通用助手（assistant）
| 维度 | 说明 |
|------|------|
| **来源** | assistant + madkg 二合一 |
| **定位** | 对话兜底 + 智能体推荐引擎 |
| **能力增强** | `useRag: true` |
| **优先级** | `priority: 1`（最低，确保兜底而非抢占） |
| **角色** | 不限 |

### 15. 🕸️ 图谱专家（graph）
| 维度 | 说明 |
|------|------|
| **核心能力** | 概念提取 + 8 种关系建模 + 层级构建 |
| **工具** | `search_concepts`、`get_graph_list`、`get_concept_relations` |
| **能力增强** | `useRag: true` |
| **角色** | 不限 |

### 16. 📦 仓库管家（repo）
| 维度 | 说明 |
|------|------|
| **核心能力** | Git 命令指导 + 提交规范检查 + 冲突解决 |
| **课程规范** | cg1-学号 / cg2-组名 / cg3-项目名 |
| **工具** | `get_my_contribution`（五维贡献度） |
| **角色** | 不限 |

### 17. 🛡️ 安全监控中心（safety）
| 维度 | 说明 |
|------|------|
| **四大监控** | 课程内容安全、系统安全、智能体质量监控、系统可靠性 |
| **审计输出** | 红/黄/绿三级风险 + 智能体质量评估表 |
| **特殊能力** | 内嵌 `PlagiarismService().listSuspicious()` 学术诚信检测 |
| **限制** | **仅管理员** |

### 18. 🔄 数字孪生（digital_twin）
| 维度 | 说明 |
|------|------|
| **来源** | virtual_student + virtual_teacher 二合一 |
| **双模式** | 学生（六维能力雷达/成长曲线/学习风格）+ 教师（教学进度映射/班级学情/风险预警） |
| **数据来源** | `TwinService().buildStudentProfile / buildTeacherProfile` 注入真实 JSON 画像 |
| **能力增强** | `useRag: true` |
| **角色** | 不限 |
