# pubspec.yaml 历史备份

本目录保存项目演进过程中的 `pubspec.yaml` 历史快照，**不参与构建**，仅供回滚参考。

| 文件 | 用途 |
|------|------|
| `pubspec_original.yaml` | 项目最早期的 pubspec |
| `pubspec_backup.yaml` | 一次重大改动前的备份 |
| `pubspec_standard.yaml` | 通用版（无 ohos 兼容修补）参考 |
| `pubspec_ohos.yaml` | 启用 HarmonyOS 模块时使用过的版本 |
| `pubspec_ohos_backup.yaml` | ohos 切换前的备份 |

> **如需恢复**：把对应文件复制到项目根，重命名为 `pubspec.yaml`，运行 `flutter pub get`。
