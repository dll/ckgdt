# 教学案例.md 模板

将本文件复制到案例项目根目录，文件名保持为 `教学案例.md`。平台添加案例时选择项目目录或直接选择该文件，会优先读取这里的信息。

## 案例名称
示例：滁州亭城 GIS 系统

## 案例全称
示例：TingChengGIS - 滁州亭文化地理信息系统

## 项目路径
可选。留空表示使用 `教学案例.md` 所在目录；也可填写相对路径，例如 `output\TingChengGIS-v1.0.5`。

## 应用类型
示例：Windows bat 启动包 / Windows EXE 应用 / Android APK 应用 / HarmonyOS HAP / Flutter Web + Go 服务

## 启动应用的方法
描述教师或平台如何启动应用。可写清楚是否会自动打开浏览器、是否需要模拟器或开发工具。

## 启动命令
可选。平台会在项目路径下执行该命令；如果留空，平台按项目类型自动识别。

## 查看应用的步骤
1. 点击案例卡片右侧启动按钮。
2. 等待应用窗口、浏览器页面或模拟器应用出现。
3. 进入首页或登录页。
4. 演示核心功能页面。

## 启动后的截图
填写相对或绝对图片路径，例如 `screenshots\home.png`。

## 应用特色内容的介绍
说明该案例适合展示的功能亮点、技术栈、教学价值和课堂提问点。

## 案例简介
补充项目背景、适用章节或实验目标。

## 仓库地址
可选。填写 Git 仓库或发布地址。
