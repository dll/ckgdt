# CKGDT 案例评比 Demo 拍摄脚本（v7 PPT 对齐版）

> **录屏目的**：配合 `docs/演示视频PPT模板v7.pptx` 16 张幻灯片，每段视频对应 PPT 1-3 张，
> 评委翻 PPT 时同步看到对应 demo 画面。
>
> **录屏工具**：OBS Studio / DevEco Studio 自带录屏 / 系统自带录屏
> **分辨率**：1920×1080 / 30 fps（移动端竖屏 1080×1920）
> **总时长目标**：6 段 × 30 秒 = 180 秒（3 分钟），可剪到 2 分钟以内
>
> **作者**：刘东良 · 安徽滁州学院 · 软件工程/智能信息系统/人工智能学习工具

## 录制前置准备

### 数据准备（拍前 1 小时）

```sql
-- 用学生账号 2023211985 登录后执行：
-- 1. 让首页菜单徽章数 > 0（点几条通知不读）
-- 2. 完成第 1 章 + 第 3 章测验（让雷达图有点）
-- 3. 提交 1 份实验报告 PDF（用 dist/ 已有学生作业 PDF）
-- 4. 等 30 秒等 grading 自动批阅完成
```

### Agent 调用准备

录之前**先用 18 个 Agent 各调用 ≥ 5 次**，让"AI 调用统计"页面排行榜满，否则截图很空：

```
小张同学：
  tutor: 问 5 次 Flutter 概念
  quiz: 生成 3 套测验
  lab: 问 2 次实验思路

小老师：
  grading: 批阅 5 份实验报告 + 3 份作品
  courseware: 生成 1 门新课程
  safety: 触发 3 次内容审查
  ethics: 触发 3 次伦理建议
```

### 网络

- **关闭学生自动同步** `sync_enabled = false`，避免录屏中 git push 弹通知
- 若需要演示同步则**保留** sync_enabled，并提前清空 sync 仓库的近期 commit 让画面干净

### 主题色

按 PPT v7 主色（深蓝 + 琥珀）：
- 默认主题色 → **深蓝**（设置页 → 主题色 → 第 1 个）
- 录第 4 段时切换到 **绿色** 或 **紫色** 演示主题切换可见效果

### 画质

- **桌面端**：1920×1080 / 60fps（用 OBS）
- **手机端**：1080×1920 / 60fps（用手机自带录屏）
- **最终输出**：H.264 / MP4 / ≤ 50 MB / 段
