# 理论教案

## 第1讲 移动应用分类与主流平台

**时间分配**：2学时（90分钟）

### 一、学生学习预期成果
能够辨析不同开发模式的技术差异，掌握主流工具链使用，了解各跨平台框架和AI编程工具的基本情况，为课程目标1提供支撑。

### 二、教学重点
技术体系分类及适用场景、开发工具链实操。

### 三、教学难点
技术选型决策逻辑。

### 四、开课提示词
1. 课程概述。
3. 课程考核。
4. 学习指导。
5. 教学要求。

### 五、学前测验（5分钟）
**测验说明**：本测验旨在检测学习者对移动应用开发基础概念和主流开发平台的了解程度。

**题目**：
1. 移动应用开发中"原生应用"的主要特征是什么？（使用平台特定的编程语言和SDK开发）
2. 微信小程序属于哪种移动应用类型？（小程序）
3. 哪个框架主要用于跨平台应用开发？（Flutter）
4. 华为鸿蒙系统的核心设计理念是什么？（多端统一生态）
5. 项目要求最高性能和用户体验，应优先考虑哪种技术方案？（原生应用开发）

**评估结果**：
- 5题全对：优秀！
- 3-4题正确：良好
- 1-2题正确：需加强
- 0题正确：建议先学习入门知识

### 六、教学内容（75分钟）

#### 1. 移动应用分类体系（20分钟）
- **原生应用**：使用平台特定编程语言和SDK开发（Android: Kotlin/Java, iOS: Swift）
- **混合应用**：Web技术与原生容器结合（Cordova、React Native、Flutter）
- **小程序**：运行在超级App内的轻量级应用（微信、支付宝）
- **多端应用**：一套代码适配多类设备（鸿蒙、Flutter）

#### 2. 主流开发平台对比（25分钟）
- **Android平台**：70%市场份额，Kotlin/Java，Google Play分发
- **iOS平台**：28%市场份额，Swift，App Store分发
- **微信小程序**：12亿+用户，MINA框架，社交+支付优势
- **华为鸿蒙**：8亿+设备，ArkUI框架，分布式架构

#### 3. 开发工具链（20分钟）
| 平台 | 开发工具 |
|------|----------|
| Android | Android Studio |
| iOS | Xcode |
| 跨平台 | Flutter、React Native、HBuilderX |
| 鸿蒙 | DevEco Studio |
| 小程序 | 微信开发者工具 |

#### 4. 技术选型基础（10分钟）
- 性能要求 vs 开发成本
- 团队技术栈匹配度
- 跨平台需求评估

### 七、课程思政（5分钟）
通过国内外移动生态竞争案例（Android/iOS vs 鸿蒙），培养科技自立自强意识与技术判断力。

### 八、学后测验（5分钟）
1. 四大移动应用类型的技术特征
2. 主流开发平台生态对比
3. 开发工具链使用方法

### 九、下节预告与作业
**下节内容**：AI编程工具（TRAE）、技术选型方法论、跨平台技术路线对比
**课后作业**：搭建各平台开发环境，创建Hello World项目

### 六、教学内容（75分钟）

#### 1. Kotlin语言基础（20分钟）
- **空安全特性**：
  ```kotlin
  var name: String? = null  // 可空类型
  val length = name?.length  // 安全调用
  val len = name?.length ?: 0  // Elvis操作符
  ```
- **Lambda表达式**：
  ```kotlin
  button.setOnClickListener { view ->
      Toast.makeText(this, "Clicked", Toast.LENGTH_SHORT).show()

- **应用场景**：
  - onCreate：UI初始化、setContentView

  ```

  GET    /api/users/123     # 获取单个用户

  PUT    /api/users/123     # 更新用户

### 六、教学内容（75分钟）

#### 1. 微信小程序概述（15分钟）
- **小程序定义**：运行在微信内的轻量级应用，无需下载安装
- **技术架构**：双线程模型（逻辑层+渲染层）、虚拟DOM
- **核心优势**：获客成本低、开发周期短、用户体验好

#### 2. MINA框架结构（20分钟）
- **文件结构**：
  ```
  pages/
    index/
      index.js      # 逻辑
      index.json    # 配置
      index.wxml   # 结构
      index.wxss   # 样式
  app.js              # 全局逻辑
  app.json            # 全局配置
  ```

#### 3. WXML语法基础（20分钟）
- **数据绑定**：
  ```xml
  <view>Hello {{name}}</view>
  <view>{{flag ? '是' : '否'}}</view>
  ```
- **列表渲染**：
  ```xml
  <view wx:for="{{items}}" wx:key="id">
    {{index}}: {{item.name}}
  </view>
  ```
- **条件渲染**：
  ```xml
  <view wx:if="{{show}}">显示内容</view>
  <view wx:else>隐藏内容</view>
  ```

#### 4. WXSS样式基础（10分钟）
- 选择器：类选择器、ID选择器、元素选择器
- 常用属性：flex布局、尺寸单位（rpx）

#### 5. 页面生命周期与路由（10分钟）
- **生命周期**：onLoad→onShow→onReady→onHide/onUnload
- **路由方式**：wx.navigateTo、wx.redirectTo、wx.switchTab、wx.navigateBack

### 七、课程思政（5分钟）

### 八、学后测验（5分钟）

| SQLite | 结构化数据 | 功能强大 | 学习成本 |
| 本地缓存 | 临时数据 | 快速访问 | 不持久 |
| 文件存储 | 大文件 | 灵活 | 管理复杂 |
