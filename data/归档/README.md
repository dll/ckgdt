# 课程归档模板目录

本目录是归档文档"套学校排版样式"的模板根。归档管线（`base_document_processor.dart`
的 `findReferenceDocx`）按以下路径查找参考样式 docx：

```
data/归档/<期>/模板/<文档名>.docx
```

其中 `<期>` 为：`期初` / `期中` / `期末` / `归档`。

## 用法

把学校原版的 Word 模板（带正确字体、页边距、页眉页脚、标题样式的空白或样例 docx）
放到对应期的 `模板/` 目录，文件名包含该文档类型的标签（如 `教学大纲.docx`、
`期末成绩分析.docx`、`归档确认表.docx`）。

放好后，归档生成的 docx 会通过 pandoc `--reference-doc` 继承该模板的排版样式，
产出接近学校原版格式的文件。

## 没有模板时

目录为空时，`findReferenceDocx` 返回 null，归档仍能生成 docx，但只用 pandoc
默认排版（字体/页边距为默认值），不继承学校样式。功能不受影响，仅样式差异。

## 注意

- 仅桌面端（Windows/macOS/Linux）+ 已安装 pandoc 时套模板生效。
- 模板文件不入库（体积大、属学校资产）；本 README 仅占位说明目录结构。
